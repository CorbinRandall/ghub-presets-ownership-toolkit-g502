G Hub Preset Toolkit — Windows

FILE TYPES (double-click these):
  .bat  = Windows batch file (runs the tool in a command window)

FIRST TIME:
  1. Double-click:  0 Setup (run once).bat
  2. You need Python 3.10+ installed (python.org) with "Add to PATH" checked

EVERY TIME (quit G Hub first!):
  1 Export from G Hub.bat   → saves profiles into the Presets\ folder
  2 Pull from Mouse.bat     → reads what's stored on the mouse hardware
  3 Import to G Hub.bat     → adds/updates profiles from Presets\ (keeps extras in G Hub)
  4 Replace G Hub with Presets.bat → G Hub = Presets folder only (removes bloat, type YES)

Your preset files live in:
  (this repo)\Presets\

You can delete, duplicate, or rename .lghub-preset.json files there like normal files.
