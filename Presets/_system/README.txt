DO NOT DELETE — G Hub system default
====================================

DONT_TOUCH_SYSTEM.lghub-preset.json is Logitech's required factory desktop
profile (default buttons, DPI, lighting). G Hub needs a profile like this.

- Do not edit or delete this file.
- Import / Replace restore it into G Hub as PROFILE_NAME_DEFAULT (Logitech's
  internal name) so it does not appear in your profile list.
- This folder is hidden from the normal Presets list on purpose.

Your profiles live in the parent Presets/ folder.
