PRESETS FOLDER
==============

Each .lghub-preset.json file here is one of YOUR G Hub profiles
(Gaming, Mac F1, Windows Work, etc.).

- Export from G Hub  → creates/updates files here
- Import / Replace    → loads these into G Hub

Factory default (do not touch):
  _system/DONT_TOUCH_SYSTEM.lghub-preset.json
  Logitech requires this; the toolkit keeps it hidden and restores it automatically.

You can delete, duplicate, or rename files in THIS folder only (not _system/).
